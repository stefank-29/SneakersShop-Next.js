# Installation

Clone repository:

```
git clone https://github.com/stefank-29/SneakersShop-Next.js.git
```

Run project:

```
npm run dev
```

### Built with:

-   React (Next.js)

### Features:

-   Main page
-   Sneakers page
-   About us page
-   Shops page
-   Adding product to cart
-   Cart page
-   Login and Register account
-   Confirm order shipping receipt
